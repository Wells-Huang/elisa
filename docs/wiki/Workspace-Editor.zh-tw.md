# 工作區編輯器 (Workspace Editor)

「工作區」分頁是您透過組合積木來設計專案的地方。

## 佈局

工作區由三個部分組成：

- **調色盤 (Toolbox)** (左側邊緣) — 您可以拖曳使用的積木類別。點擊類別名稱即可展開，然後將積木拖到畫布上。
- **畫布 (Canvas)** (中央) — 您排列積木的主要編輯區。積木會垂直組合在一起。
- **側欄 (Sidebar)** (右側邊緣) — 包含檔案操作、技能、規則、門戶、範例以及說明的按鈕。

## 畫布控制

| 動作 | 如何執行 |
|--------|-----|
| 平移畫布 | 在畫布空白處點擊並拖曳 |
| 縮放 | 使用滑鼠滾輪，或使用右下角的「+/-」控制按鈕 |
| 對齊網格 | 積木會自動對齊 20px 的網格 |
| 刪除積木 | 將積木拖到右下角的垃圾桶，或選取積木後按下 Delete 鍵 |
| 復原/重做 | Ctrl+Z / Ctrl+Y (Mac 上為 Cmd+Z / Cmd+Shift+Z) |

縮放範圍為 0.3 倍至 3 倍。

## 側欄按鈕

| 按鈕 | 作用 |
|--------|-------------|
| **資料夾 (Folder)** | 從電腦上的資料夾開啟工作區。選定資料夾後按鈕會變為綠色。 |
| **開啟 (Open)** | 開啟一個 `.elisa` nugget 檔案 |
| **儲存 (Save)** | 儲存您的專案。若已設定工作區資料夾，則儲存設計檔案至該處；否則將下載 `.elisa` 檔案。 |
| **技能 (Skills)** | 開啟 [技能 (Skills)](Skills.zh-tw.md) 編輯器以建立可重複使用的提示詞片段 |
| **規則 (Rules)** | 開啟 [規則 (Rules)](Rules.zh-tw.md) 編輯器以為代理設定護欄 |
| **門戶 (Portals)** | 開啟 [門戶 (Portals)](Portals.zh-tw.md) 編輯器以連接外部工具與硬體 |
| **範例 (Examples)** | 瀏覽並載入預設的範例專案 |
| **說明 (Help)** | 使用 Elisa 的快速參考指南 |

## 主要分頁

標題列有兩個主要分頁：

- **工作區 (Workspace)** — 積木編輯器（始終可用）
- **任務控制中心 (Mission Control)** — 任務圖、僕從小隊與講述者饋送（僅在建置期間啟用）

當您按下 GO 時，視圖會自動切換到「任務控制中心」。您隨時可以切換回「工作區」，但在建置期間畫布是唯讀狀態（會顯示「正在建置中...」的覆蓋層）。

建置期間還會出現額外分頁：
- **代理 (Agents)** — 全寬度的代理卡片與通訊饋送
- **任務 (Tasks)** — 全寬度的互動式任務圖

## 自動儲存

您的工作區、技能、規則與門戶會在每次變更時自動儲存到瀏覽器的 `localStorage`。當您重新開啟 Elisa 時，所有內容都會恢復到您上次離開時的樣子。不需要手動儲存。

詳見 [儲存與讀取 (Saving and Loading)](Saving-and-Loading.zh-tw.md) 獲取更多關於檔案儲存與讀取的選項。
