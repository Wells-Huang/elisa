# Elisa 使用者手冊

Elisa 是一款視覺化程式設計工具，您可以透過組合積木來描述想要構建的專案，然後由 AI 代理為您實作。它專為孩子（10 歲以上）及程式設計初學者所設計。

您使用積木設計專案，點擊 GO，即可即時觀看您的 AI 僕從團隊進行任務規劃、撰寫程式碼、執行測試及部署。
